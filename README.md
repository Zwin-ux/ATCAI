# ATCAI

AI air-traffic-control simulation and visualization prototype. The active docs and implementation live in `ATC/`.

## What is here

- `ATC/README.md`, `QUICKSTART.md`, and `ARCHITECTURE.md` - main project documentation.
- `ATC/launch_dashboard.py`, `launch_echo_tower.py`, and related launch scripts - demo entry points.
- `ATC/bluesky_adapter/`, `atc_environment/`, `configs/`, and `examples/` - simulation and integration areas.
- `ATC/VISUALIZATION_PLAN.md` and completion notes - build history and feature notes.

## Start point

```bash
cd ATC
python launch_dashboard.py
```

For setup details, use `ATC/QUICKSTART.md` before running the demos.

## Status

Prototype. The root README is intentionally short so the GitHub page points to the real implementation docs without overselling the current state.
